# Requirements-Response Strategy Agent

> **Prerequisite:** This agent requires a project with uploaded RFP and proposal documents. See [RFP Advisor Setup](/docs/rfp-advisor) for how to create a project and upload documents.

The Requirements-Response Strategy Agent is the most comprehensive analysis agent. It performs a 13-agent analysis across 5 phases, comparing your proposal against RFP requirements and developing a complete win strategy.

---

## What It Does

Get a complete strategic picture:

- **Map requirements to responses** — What's covered, what's missing
- **Score response quality** — Depth, evidence, clarity, persuasiveness
- **Verify compliance** — Pass/fail audit of mandatory requirements
- **Develop win themes** — Top 5 themes with supporting evidence
- **Create improvement roadmap** — Prioritized actions with timelines
- **Build win strategy** — Cohesive approach to winning

---

## The 5-Phase Pipeline

This agent runs 13 sub-agents across 5 phases:

### Phase 1: Intelligence Gathering

| Agent | What It Does |
|---|---|
| Requirements Intelligence | Catalogs all RFP requirements |
| Response Intelligence | Catalogs all proposal elements |
| Client Intent Analysis | Infers client priorities and mindset |

### Phase 2: Gap & Compliance Analysis

| Agent | What It Does |
|---|---|
| Requirements Coverage | Maps requirements to responses |
| Quality & Depth Analyzer | Scores each response element |
| Compliance Verification | Audits mandatory requirements |

### Phase 3: Strategic Analysis

| Agent | What It Does |
|---|---|
| Competitive Positioning | Analyzes strengths vs. competitors |
| Win Theme Strategist | Develops top 5 win themes |
| Risk & Mitigation | Creates risk register with mitigations |

### Phase 4: Recommendations

| Agent | What It Does |
|---|---|
| Improvement & Enhancement | Creates prioritized improvement roadmap |
| Win Strategy Architect | Synthesizes cohesive win strategy |

### Phase 5: Reporting

| Agent | What It Does |
|---|---|
| Executive Report Compiler | Structures the final report |
| Quality Assurance Review | Checks accuracy and consistency |

---

## What You Get

**Executive Dashboard:**
- Win probability
- Coverage percentage
- Compliance status
- Top 3 immediate actions

**Coverage Matrix:**
- Each requirement mapped to response
- Coverage levels: Full, Partial, Weak, Missing
- Gap analysis highlighting what's missing

**Quality Scores:**
- Depth (1-10)
- Evidence (1-10)
- Clarity (1-10)
- Persuasiveness (1-10)
- Top 5 strongest and weakest responses

**Compliance Audit:**
- Pass/Conditional/Fail for each mandatory requirement
- Go/No-Go assessment

**Win Themes:**
- Top 5 themes to emphasize
- Supporting evidence for each
- How to apply in the proposal
- Competitive counter-positioning

**Risk Register:**
- Risks across 5 categories
- Probability and impact ratings
- Mitigation strategies

**Improvement Roadmap:**
- Priority 1: Must fix before submission
- Priority 2: Should fix if time permits
- Priority 3: Nice to have
- 48-hour critical checklist

**Win Strategy:**
- Strategic pillars
- Execution roadmap (48 hours, 1 week, final)
- Stakeholder engagement plan

---

## When to Use It

| Situation | Use Requirements-Response Strategy? |
|---|---|
| Have both RFP and draft proposal | Yes — this is the ideal use case |
| Pre-submission strategic review | Yes |
| Need complete gap analysis | Yes |
| Only have RFP, no proposal yet | No — use Requirements Review first |
| Only have proposal, no RFP | No — use Response Review |

---

## Input Requirements

This agent requires BOTH types of documents:

**RFP Documents:**
- Main RFP
- Amendments
- Attachments

**Proposal Response Documents:**
- Your draft proposal
- Technical approach
- Management approach
- All response sections

Classify documents appropriately in your project for best results.
