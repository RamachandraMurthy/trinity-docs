# Technical & Planning Agents

> **Prerequisite:** These agents require a project with uploaded RFP and proposal documents. See [RFP Advisor Setup](/docs/rfp-advisor) for how to create a project and upload documents.

This page covers several specialized agents for technical and planning aspects of proposals: **Technical Architecture**, **Timeline Planning**, **Stakeholder Analysis**, and **Storytelling Narrative**.

---

## Technical Architecture Agent

Validates and enhances technical solution designs in your proposal.

### What It Does

- **Assess architecture** — Evaluate current technical design
- **Review technology stack** — Appropriateness of chosen technologies
- **Validate integration** — How components work together
- **Check security** — Security architecture validation
- **Plan implementation** — Technical delivery roadmap

### The Pipeline (5 agents)

Document Understanding → Technical Requirements → Solution Architecture → Technical Validation → Implementation Planning

### What You Get

- Architecture assessment with recommendations
- Technology stack review
- Infrastructure and integration analysis
- Security architecture validation
- Technical implementation roadmap

---

## Timeline Planning Agent

Develops project timelines and resource plans.

### What It Does

- **Analyze timeline** — Critical path and dependencies
- **Plan resources** — Staffing and resource requirements
- **Estimate budget** — Cost by phase and work package
- **Assess schedule risk** — Risk register with contingencies
- **Optimize delivery** — Timeline with buffers and recovery options

### The Pipeline (5 agents)

Document Understanding → Timeline Analysis → Resource Requirements → Risk Assessment → Implementation Planning

### What You Get

- Critical path analysis with milestones
- Resource allocation and staffing plan
- Budget estimates by phase
- Schedule risk register
- Optimized timeline with contingencies

---

## Stakeholder Analysis Agent

Identifies stakeholders and develops engagement strategies.

### What It Does

- **Identify stakeholders** — Who matters in the decision
- **Map influence** — Power and interest levels
- **Develop personas** — Understanding key decision makers
- **Plan engagement** — Tailored strategies per stakeholder
- **Guide communication** — Messaging and timing

### The Pipeline (5 agents)

Document Understanding → Stakeholder Identification → Stakeholder Analysis → Engagement Strategy → Communication Planning

### What You Get

- Stakeholder register
- Power/interest matrix
- Key stakeholder personas
- Engagement strategies by stakeholder
- Communication plan with messaging

---

## Storytelling Narrative Agent

Develops compelling narratives and emotional connections in your proposal.

### What It Does

- **Identify value propositions** — Core messages
- **Build narrative arcs** — Compelling story structure
- **Create emotional connection** — Engagement points
- **Apply persuasion** — Psychological elements
- **Ensure consistency** — Unified messaging framework

### The Pipeline (5 agents)

Document Understanding → Value Proposition → Narrative Development → Psychological Impact → Executive Report

### What You Get

- Core value propositions
- Narrative arc using Hero's Journey framework
- Emotional engagement points
- Persuasion elements
- Consistent messaging framework

---

## When to Use These Agents

| Need | Which Agent? |
|---|---|
| Validate technical solution | Technical Architecture |
| Develop project timeline | Timeline Planning |
| Understand decision makers | Stakeholder Analysis |
| Improve proposal narrative | Storytelling Narrative |
| Complete proposal review | Use with other RFP agents |
